# Pinnacle Thoughts

Static marketing site template (HTML + CSS + JS).

## Preview locally
- Open `index.html` in your browser for quick checks.
- Or run a simple static server:
  - Python 3: `python -m http.server 8000`
  - Node: `npx http-server -c-1 .`

## Git
Repository initialized with Git. To add a remote and push:
```bash
# replace <url> with your remote URL
git remote add origin <url>
git push -u origin main
```
